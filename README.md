# 🌐 HTML Practice → Portfolio Foundation

This repository represents my early exploration into HTML and basic web development.

While it started as simple practice, it evolved into the foundation for building a personal portfolio website and understanding how front-end systems are structured.

---

## 🧠 Purpose

This project was used to learn and experiment with:

- HTML structure and semantic elements
- Linking between pages
- Image handling and layout
- Basic CSS styling
- Navigation and anchors
- External links and downloads

It serves as a **learning sandbox** and a **starting point for more advanced web projects**.

## ▶️ How to Run

1. Clone or download the repo
2. Open `index.html` in a browser

---

## 🔍 Features Demonstrated

- Internal navigation (`#anchors`)
- Multi-page navigation (`about.html`)
- Image embedding and styling
- Downloadable assets
- External links (Google, email)
- Inline CSS styling
- Basic responsive meta setup

---

## ⚠️ Known Limitations

- No responsive design beyond basics
- Inline CSS instead of modular stylesheets
- No JavaScript interactivity
- No layout system (Flexbox/Grid)
- No consistent visual theme
